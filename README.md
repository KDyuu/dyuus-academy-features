# Dyuu's Academy Features

Dyuu's Academy Features est un mod Fabric spécialement conçu pour Cobblemon Academy 2.0. 
Il apporte des interfaces graphiques (GUI) entièrement personnalisées et optimisées pour améliorer l'expérience de jeu.

## Fonctionnalités principales
- Shop personnalisé : Interface moderne avec pagination et PokéDollars
- Daycare : GUI intuitive pour la gestion des Pokémon (pas encore implémenter)
- Terastalisation : Interface dédiée pour changer les types téra (pas encore implémenter)
- Battle Factory : Écran custom pour les combats organisés (pas encore implémenter)
- Système d'argent intégré : PokéDollars avec achat/vente d'items
- Autres GUI en cours : Plus de fonctionnalités arrivent !

## Dépendances requises :
Minecraft 1.21.x
Fabric Loader
Fabric API
Modpack Cobblemon Academy 2.0
Datapack custom me contacter pour l'avoir

## Installation
Pas de guide pour le moment. Installer le mods et le mettre dans le dossier mods ne suffit pas.

## Système PokéDollars
Commandes :
/shop
/shop reload

Les prix s'affichent dans les tooltips des items.
le mod génère un fichier json pour modifier le contenue du shop dans le dossier config de minecraft
